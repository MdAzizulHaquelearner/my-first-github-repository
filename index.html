How to use Github

GitHub, Inc. is an AI-powered developer platform that allows developers to create, store, and manage their code. It uses Git software, providing the distributed version control of Git plus access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project. Wikipedia

Github command line

  • git init
  • git add .
  • git commit -m "first commit"
  • git branch -M main
  • git remote add origin https://github.com/MdAzizulHaquelearner/my-first-repository.git
  • git push -u origin main
